## 2. 마크다운 태그(문법) 

### 2.1 제목

```
# 제목 1 <H1>
## 제목 2 <H2>
### 제목 3 <H3>
#### 제목 4 <H4>
##### 제목 5 <H5>
###### 제목 6 <H6>
```

> **적용 예시**
___
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6

```
제목 1
======

제목 2
------
```

> **적용 예시**
___

제목 1
======

제목 2
------

### 2.2 목록
- **순서 O**

```
1. 목록 1
2. 목록 2
3. 목록 3

1. 목록 1
3. 목록 2
5. 목록 3
```

> **적용 예시**
___
1. 목록 1
2. 목록 2
3. 목록 3
___
1. 목록 1
3. 목록 2 (3. 작성)
5. 목록 3 (5. 작성)

```
* 가위
  * 바위
    * 보

+ 가위
  + 바위

  - 바위
    - 보

   + 바위

### 2.3 강조

```
<이탤릭체>
*이탤릭체*
_이탤릭체_

<볼드체>
**볼드체**
__볼드체__

<이탤릭체 + 볼드체>
***혼합***
___혼합___
*__혼합__*

<취소선>
~~취소선~~
```

> **적용 예시**
___

- *이탤릭체*
- _이탤릭체_

- **볼드체**
- __볼드체__

- ***혼합***
- ___혼합___
- *__혼합__*

- ~~취소선~~

### 2.4 인용구 (BlockQuote)
 ```
 > 인용 1
 >> 인용 2
 >>> 인용 3
 
 <이 안에서는 다른 마크다운 문법 사용 가능>
> ## 인용
> _인용 2_
> * 인용 3
> 
> **인용 4**
 ```
 
> **적용 예시**
___

> 인용 1
>> 인용 2
>>> 인용 3
___

> ## 인용
> _인용 2_
> * 인용 3
> 
> **인용 4**

### 2.8 링크

- **일반적인 링크 사용**

```
[Title](link)

[NAVER](https://www.naver.com)
```

> **적용 예시**
___

[NAVER](https://www.naver.com)

- **자동 링크 사용**

```
일반적인 URL 또는 이메일 주소

<link>

<https://www.naver.com>    
<seohyuniiii@example.com>
```

> **적용 예시**
___

<https://www.naver.com>    
<seohyuniiii@example.com>

- **참조 링크 사용**

```
[link keyword][id]
(한줄 띄어쓰기)
[id]: URL "설명 문구 (생략가능)"

링크를 [NAVER][naver link] 클릭하세요

[naver link]: https://www.naver.com "네이버"
```
 
> **적용 예시**
___

링크를 [NAVER][naverlink] 클릭하세요

[naverlink]: https://www.naver.com "네이버"
